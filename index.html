About

Zero Office is an internal workflow tool used by the creator to manage owned or authorized short-form video clips. The system helps prepare clips, review captions, check compliance status, and publish approved videos to connected creator accounts.

TikTok Integration

The TikTok integration uses TikTok OAuth so the account owner can connect their own TikTok account. The Content Posting API may be used to publish prepared MP4 videos only after manual creator approval inside the workflow.

Zero Office does not post to third-party accounts, does not scrape TikTok, and does not publish without explicit creator approval.

Data Use

The app stores only the minimum technical data required for the workflow, such as OAuth token data, upload status, clip mapping, timestamps, and basic metadata needed for creator-side operations and analytics.
